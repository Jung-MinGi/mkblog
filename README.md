<h2>블로그형 개인 프로젝트</h2>(❗프리티어 기간만료로 서버는 다운되었습니다)

- 배포 환경: AWS

- 기술 스택
  - Java17
  - SpringBoot
  - MySql
  - Thymeleaf

<h2>구현 기능</h2>

- 게시글 CRUD
- 메인페이지 글 목록 페이징처리
- 섬머노트를 사용하여 글 작성 및 편집 기능을 구현
